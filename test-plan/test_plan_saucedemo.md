# Test Plan – SauceDemo

## Objective
Verify that core functionalities of the SauceDemo web application work as specified in the defined requirements.

## Test Scope
Login, product overview, cart, checkout (happy path).

## Test Approach
Manual functional testing based on defined requirements using basic black-box test design techniques.

## Test Environment
- Browser: Google Chrome (latest stable version)
- OS: Windows 11 (latest stable version)

## Entry Criteria
Application is accessible.

## Exit Criteria
All planned test cases are executed and critical defects are documented.
